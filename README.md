# 🚀 Fzlwhyc OpenClash Templates

**沉浸式排版与智能调度配置模板 | 为 OpenClash 玩家量身定制**

绝大多数机场自带的订阅配置，只是把所有节点粗暴地堆砌在一起，不仅面板混乱，而且毫无逻辑可言。本项目的初衷并非改变 OpenClash 的底层机制，而是**深度重构你的网络交互体验**，消除选择困难症，让合适的节点在合适的时候，出现在最顺手的位置。

## ✨ 核心特性升级

### 1. 🎯 业务级“智能优先级”分流
告别无脑的“全局测速”。本模板根据各大平台的版权与风控逻辑，为你把各组的“天然最优解”排在了首位：
* **👽 AI 服务**：优先展示 `[日/美/台/韩]`，底层物理剔除 `[香港]` 测速组，彻底杜绝因节点漂移导致 ChatGPT/Claude 封号。
* **🎬 流媒体**：优先展示 `[港/台]` 节点，保证 Netflix/Disney+ 等最高解锁率及原生中文字幕。
* **📡 Telegram**：优先展示 `[新/港]` 节点，直连亚洲核心数据中心，图文秒开。
* **📘 下载/GitHub**：优先展示 `[新/日/韩]` 节点，吃满亚太海底光缆极限大带宽。

### 2. 🎛️ 交互重构与“精准锁定”机制
* **👁️ 视觉降噪**：将不可点选操作的“自动测速挡”全部沉底隐藏，常用的“手动挡”提前，首屏干净清爽。
* **🌍 区域锁定**：【国外总模式】作为全局中枢，功能等同于“指定大区”。在此锁定任意地区后，全站流量会自动在该地区内择优而行。
* **📦 冷热分流哲学**：摒弃将所有节点塞入一个大下拉框的臃肿做法。将“同地区多节点”划入【🔥 热门】，将“单节点小众地区”划入【❄️ 冷门】。完美展现机场特色版图，单节点不再被淹没。
* **🎯 节点精准狙击**：弥补了 OpenClash 常规配置的一大痛点。通过下放至各业务底部的【手动组】，你可以随时接管测速权，将某一应用死死锁定在特定的单一节点上。

## 🛠️ 极简使用指南

本模板通过 Subconverter（订阅转换）实现，只需在 OpenClash 中简单配置即可应用。

### 第一步：获取模板直链
复制以下链接（这是本仓库模板的 Raw 直链）：
> `https://raw.githubusercontent.com/Fzlwhyc/OpenClash-Templates/main/Fzlwhyc_OpenClash.ini`

### 第二步：在 OpenClash 中应用
1. 进入路由器的 OpenClash 后台面板。
2. 导航至 **配置订阅** -> 选择你现有的机场订阅配置 -> 点击 **编辑**。
3. 在编辑页面，找到 **订阅转换模板 (Template)** 或直接在 **自定义配置 (Custom Config)** URL 尾部添加参数。
4. 将第一步复制的直链填入。
5. 保存配置，并点击 **更新订阅**。

🎉 **刷新你的 YACD 面板，享受极致清爽的冲浪体验吧！**

## 📝 兼容性说明
本模板的 `[🔥 热门-手动]` 与 `[❄️ 冷门-手动]` 已内置国际通用的中英文及双字母缩写宽容正则表达式。无论是主流机场还是小众服务商，只要节点命名包含基本地理特征（如 香港、HK、Hong、日本、JP 等），即可实现 100% 精准抓取与分类。
